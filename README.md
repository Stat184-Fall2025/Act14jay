# Activity 14 – First Quarto (QMD) Project

## Project Overview
This repository contains my submission for **STAT 184 – Activity #14**.  
The goal of this activity is to practice creating a **reproducible analysis report** using **Quarto (.qmd)**, including narrative text, code chunks, and a rendered PDF.

## Data
The dataset used in this activity is stored in `data.csv` and was provided as part of the course materials for Activity #14.

## Plan
The project plan is documented in `plan.md` and outlines:
- The goal of the activity
- The steps to set up and maintain the GitHub repository
- The steps to complete the Quarto analysis and render the PDF

## Repository Structure
